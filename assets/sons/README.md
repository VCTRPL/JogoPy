# Sons

Pasta para efeitos sonoros e trilhas do jogo.

## Conteúdo esperado

- `musica.ogg`: trilha de fundo tocada em loop durante a partida (opcional).
  Se o arquivo não existir, o jogo roda normalmente em silêncio.

## Como adicionar música

Baixe um arquivo de áudio com licença livre (sugestões: Kenney Assets, OpenGameArt,
itch.io) e salve como `musica.ogg` nesta pasta. Registre a origem e a licença no
README principal do projeto.

## Recomendações

- Use formatos leves: `.ogg` para música e `.wav` para efeitos curtos.
- Normalize o volume para evitar diferenças bruscas entre arquivos.
- Documente a origem dos áudios quando forem externos.
